# 水獭角色规范

本目录用于维护“水獭星球”主角水獭的统一视觉基准。

当前主参考图：

- [otter-main-reference.png](./exports/otter-main-reference.png)

后续所有水獭角色图、页面立绘、头像、姿态图、表情图、剧情演出图，都应以这张图为基础延展，不再回到旧的手工绘制占位造型。

## 角色定位

- 角色身份：`水獭星球` 主角水獭
- 面向用户：3-8 岁儿童
- 气质关键词：温暖、可信任、友好、陪伴感、低攻击性、可爱但不过度幼稚
- 使用场景：首页主视觉、领养页角色初见、森林冒险陪伴角色、后续头像/贴纸/剧情演出

## 视觉基准

### 1. 脸部结构

- 头部整体偏大，接近圆角椭圆，头身比明显偏向儿童友好风格
- 耳朵为圆形小耳，左右对称，位置偏高
- 眼睛为大号深色椭圆眼，保留高光，避免写实瞳孔细节
- 鼻子为大号椭圆深棕色鼻头，位于面部中心，是识别重点
- 嘴部很小，默认轻微张嘴，带一点橙红色舌头，整体情绪偏开心
- 胡须明显、细长、左右对称，是重要识别特征，不要删掉
- 眉部不是写实眉毛，而是浅色小弧形眉斑，表达温和情绪

### 2. 身体比例

- 站立姿态为直立式卡通水獭
- 身体是整体圆润的长椭圆形，不要做成瘦长或尖锐轮廓
- 四肢短小，手掌和脚掌简化处理，避免锋利爪感
- 尾巴粗而长，从身体后侧自然延伸，保持圆滑弧线
- 脖颈不强调结构，头身连接自然柔和

### 3. 毛色与配色

- 主体毛色：暖棕色、焦糖棕、浅橙棕方向
- 面部口鼻区、胸口、腹部为浅奶油色/米白色
- 鼻头、眼周深色区域、手脚阴影可使用更深的棕色
- 舌头保留柔和橙红色点缀
- 整体配色必须偏暖，不能偏灰、偏冷、偏脏

建议遵循的配色倾向：

- 主毛色：蜂蜜棕、焦糖棕、暖木色
- 辅助浅色：奶油白、浅米色、柔和杏色
- 深色点缀：深棕、巧克力棕

## 造型原则

- 轮廓圆润优先，不要尖嘴、尖耳、尖下巴
- 眼神要开放友好，不要做成冷淡、狡猾或攻击性表情
- 默认状态是“微笑陪伴型”，而不是夸张搞怪型
- 即使做动作变化，也要保留“安全感”和“可亲近感”
- 尽量维持当前参考图的正面识别度，特别是：
  - 大鼻子
  - 奶油色口鼻区
  - 长胡须
  - 圆头圆耳
  - 暖棕主体

## 表情方向

推荐优先扩展这些表情：

- 默认微笑
- 开心
- 好奇
- 鼓励
- 认真倾听
- 轻微惊讶
- 温柔安慰

不建议优先扩展这些方向：

- 愤怒
- 凶狠
- 夸张邪笑
- 高强度哭泣
- 过度嘲讽或戏谑

## 姿态方向

后续 `poses/` 目录建议优先补这些姿态：

- 站立默认姿
- 招手欢迎
- 指向前方
- 双手鼓励
- 倾听孩子说话
- 小跑/探险前进
- 举起月亮碎片
- 开心庆祝

## 页面落地建议

- 首页：适合使用完整立绘，突出“欢迎进入冒险世界”
- 领养页：适合使用正面站立或轻微招手姿态，强化“等待被命名”
- 森林冒险页：适合使用陪伴型站姿、倾听型姿态、轻动作姿态
- 对话面板：优先使用头像裁切版或半身版，保证表情清晰

## 文件放置约定

- `concept/`：角色设定稿、改版草图、设计探索图
- `poses/`：不同动作、不同镜头姿态图
- `ui/`：头像、徽章、小卡、气泡旁头像
- `exports/`：前端直接使用的最终 png/webp/svg

## 禁止项

- 不要改成写实水獭比例
- 不要改成攻击性猛兽脸
- 不要弱化鼻子、胡须、奶油色口鼻区这些核心识别特征
- 不要把整体颜色改成冷灰、蓝灰、脏棕
- 不要把四肢和尾巴画得过细，避免失去儿童产品的圆润感
- 不要在不同页面里频繁更换基础脸型

## 当前结论

从现在开始，`otter-main-reference.png` 是项目内的主角色视觉锚点。

如果后续要升级角色风格，应遵循：

1. 先在 `concept/` 中保留探索稿
2. 确认没有偏离当前角色识别特征
3. 再输出到 `exports/`
4. 最后替换页面中的统一角色引用
