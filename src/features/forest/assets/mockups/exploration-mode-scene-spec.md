# 探索模式场景说明

主背景：

- `backgrounds/forest-exploration-first-person-inverted-canopy.png`

当前探索模式的视觉基准采用第一人称视角：

- 玩家视角沿着发光花藤道路向前推进
- 水獭不是固定站在画面中央，而是作为陪跑伙伴和引导者存在
- 树冠是倒悬的，空间方向需要保持“向前穿梭但上方翻转”的奇幻感
- 月光瀑布、星星灯牌、晶体、漂浮石块共同形成路径引导

本轮补充的前端可用素材：

- `effects/forest-glow-star.svg`
- `effects/forest-path-ribbon.svg`
- `effects/forest-waterfall-column.svg`
- `props/forest-floating-crystal.svg`

这些素材当前定位为：

- 页面演示期的气氛增强素材
- 后续 Babylon 场景搭建前的视觉占位
- 设计和前端同步探索模式镜头语言的参考件
