# 冒险页第一阶段说明

## 本轮目标

根据当前决策，本轮暂不接入训练模块、手势识别和语音引擎，而是优先搭建冒险页本身，让它成为后续剧情系统的主容器。

## 本轮完成内容

### 1. 修复页面文本

- 修复冒险页内出现的中文乱码
- 统一冒险页文案为中文叙事风格

### 2. 冒险页结构升级

将原本偏展示型的页面，升级为具备剧情容器属性的页面，包含以下区域：

- 场景舞台区
- 当前目标卡
- 章节状态 HUD
- 星星与月光值状态
- 主线推进节点
- 页面承载模式说明
- 场景分层说明
- 关键 NPC 入口卡
- 后续接入点说明

### 3. 场景视觉增强

在冒险页中新增以下视觉元素：

- 月光瀑布
- 发光粒子
- 水獭角色占位
- 目标提示卡
- 更完整的森林 HUD

## 为什么先做这一层

这样做的好处是，后续无论接剧情状态机、NPC 出场、结算流程，还是接真实 3D 渲染，都有一个稳定的页面骨架可继续扩展，而不会把交互、剧情和视觉同时混在一起开发。

## 下一步建议

下一步优先推进以下内容，而不是训练模块：

1. 剧情状态机基础版
2. NPC 出场与对话面板
3. 冒险页章节切换
4. 奖励与任务状态同步
5. 真月亮揭晓页内流程
